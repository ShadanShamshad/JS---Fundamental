Variables

Variables are containers for pieces of data,they are used to store the information. Variables store data of any data type that can be used throughout a program.

JavaScript Variables are the building blocks of any programming language. In JavaScript, variables can be used to store reusable values. The values of the variables are allocated using the assignment operator("=").

The let keyword was added to JavaScript in 2015.

With let keyword we declare in the following way:-

let button ="PW-Skills"
