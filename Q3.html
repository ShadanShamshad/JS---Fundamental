Five primitive data types in JavaScript

  1. String : Combination of alphanumeric characters wrapped in either single or double quotes or in backticks. Strings can include any number, letter or symbol.

    For Example:-
    const name = 'ram'; (Single Quote)
    const name1 = "hari"; (Double Quotes)
    const result = `The names are ${name} and ${name1}`; (Backticks)

  2. Number : Any number in JavaScript is the Number type, including floats and decimals. Some langu1ges have separate types for floats and integers. JavaScript does not. Numbers are not wrapped in quotes.

    For Example:-
    const number1 = 3;
    const number2 = 3.433;

  3. Null : Null means nothing or empty value. It is often used to indicate that a variable or property has no value.

    Note: null is not the same as NULL or Null.

    For Example:-
    const number = null;

    The code above suggests that the number variable is empty.

  4. Undefined : undefined is a special value that indicates that a variable or property has been declared but has not been assigned a value

    For Example:-

    let name;
    console.log(name); // undefined

    It is also possible to explicitly assign a variable value.
    let name = undefined;
    console.log(name); // undefined

    Note: It is recommended not to explicitly assign undefined to a variable. Usually, null is used to assign 'unknown' or 'empty' value to a variable.

  5. Boolean : A boolean is a true or false value. This data type represents logical entities. Boolean represents one of two values: true or false. It is easier to think of it as a yes/no switch.

    For Example:- const dataChecked = true;
    const valueCounted = false;
